[문제 링크](https://www.acmicpc.net/problem/1550)

# 풀이
1. char형으로 입력을 받는다
2. 만약 알파벳(아스키코드)이라면 9(아스키코드 = 57)보다 크므로 조건문으로 분류
3. 분류된 알파벳을 55를 빼주면 될것이다.(A가 65부터 시작이므로)

# 첫번째 시도
한글자만 입력되는줄 알았는데 여러 글자가 들어간다.

# 두번째 시도
char형으로 받지 않고 string형으로 받은 뒤 위에 적혀있는 풀이대로 적용
각 자릿수는 16의 n승이므로 string의 길이에 맞춰 빼서 적용한다.
